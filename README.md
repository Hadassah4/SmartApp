SmartApp - Slim geregeld elke dag opnieuw. 

1. In dit project:
Welkom bij mijn SmartApp. De code die ik heb gemaakt leest dagelijke informatie zoals de buitentemperatuur, binnentemperatuur,
het aantal personen thuis en de hoeveelheid neerslag die dag af. Op basis hiervan beslist het systeem die ik heb gemaakt zelf of
de CV-ketel aan moet, of de ventilatie moet draaien en of de planten water nodig hebben.
Ook heb ik de het bestand 'Weerstation' gemaakt. Hierin kan je de aankomende 7 dagen de temperatuur, windsnelheid en luchtvochtigheid
invoeren. Als output krijg je een weerrapport te zien.

De gebruiker van mijn Smartapp kan in het Menu zien wat hij met de Smartapp kan doen.
Dit zijn de opties:
1. Het aantal dagen in de dataset bekijken  
2. Automatisch alle actuatoren laten berekenen  
3. Handmatig waarden overschrijven  
4. Stoppen
